# catGreeting

### Prerequisites
    node.js = 14.15.4
    npm = 6.14.10

### Install
 ``` npm install ```

### Run Locally

``` node server.js ```

### input parameters
width, height, color, size, and fileName are not mandatory
```
    {
        greetings: 'Hey there shamila',
        width: 400, 
        height: 300,
        color: 'Pink', 
        size: 100,
        fileName: 'mergedImage.png'
    }
```
